# q2a-tagextra

<b>Plugin Name:</b>  TagExtra <br>
<b>Price:</b> 30 USD (contact me to buy) <br>
<b>Plugin Description:</b> This is premium paid plugin that structures tags page like stackoverflow. It allows admins add/edit descriptions to each tags, and allows users to follow tags and structure their feeds with followed-tags questions.<br>
<br>
<ul class="first">
	<b>Plugin Functionality:</b>
	<li>Plugin adds a follow button for each tag. So, users can follow/unfollow tags.</li>
	<li>Plugin adds a search filter at tags page where you dynamically search tags.</li>
	<li>Plugin adds an add/edit button for each tag which is visible only for admins. The button adds/edits description for each tag.</li>
	<li>Plugin adds "My Feed" tab onto question list pages. "My Feed" tab shows the list of questions with only tags that the user follows.</li>
	<li>Admin can determine which user levels are authorized to add/edit tags descriptions.</li>
